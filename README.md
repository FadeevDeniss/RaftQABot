## Запуск

1. Извлеките файлы проекта в одну папку.
2. Создайте виртуальное окружение.
3. Установите зависимости ```pip install -r requirements.txt```
4. Скопируйте содержимое папки extra в 'venv/Lib/site-packages/openai'.
5. Создайте файл .env и укажите переменные 'OPENAI_API_KEY' и 'TG_API_KEY'.
6. Запустите бота командой ```python run.py```
   - В начале запуска выполняется парсинг сайта по указанным
     настройкам, это может занять до 10 минут с текущими настройками из-за 
     ограничений API. В результате в папке text в корне проекта
     появится файл embeddings.csv для создания вложений.
   - Вопрос можно задать с помощью команды '\ask'
